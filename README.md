### Please follow the steps below :

    1. Install node js from https://nodejs.org/en/download
    2. Check node version in terminal using

        
        node --version

    3. Node go to the project path in terminal using cd path of the project. (Get path from folder)
        ex. cd  /users/projects/futurestack
    4. Check node version again
         
         node --version

    5. Install packages but this time just run below command and it will take package names from package.json automatically
        
        npm i
        
    6. Run the server 

        node app.js
    
    7. Go to the home page

        http://localhost:4000/

    8. If facing any error put message on the group with error you are facing. May be screenshot of error will help to debug faster.
